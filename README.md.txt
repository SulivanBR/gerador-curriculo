# Gerador de Currículo em PHP

Sistema que permite ao usuário preencher dados pessoais e gerar um currículo em formato PDF.  
Desenvolvido com **PHP**, **Bootstrap**, **jQuery** e **Dompdf**.

---

## Funcionalidades
- Inclusão de dados pessoais  
- Cálculo automático da idade  
- Adição e remoção de experiências profissionais  
- Geração de currículo em PDF formatado  
- Layout responsivo

---

1. Instale o **XAMPP** e inicie o **Apache**.
2. Copie a pasta do projeto para:
